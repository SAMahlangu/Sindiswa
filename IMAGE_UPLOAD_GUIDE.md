# Image Upload Guide for Admin

## How to Add Images to Services

Your admin dashboard now supports adding images to each service. Here's how to use it:

### Adding Images When Creating a Service

1. Go to the **Admin Dashboard** → **Services** tab
2. Fill in the service details:
   - Service Name
   - Price
   - Duration
   - Description
3. **Add Image URL** - Paste the URL of your image:
   - You can use images from **Pexels** (https://www.pexels.com) - completely free!
   - Or **Unsplash** (https://www.unsplash.com)
   - Or upload to any hosting service and paste the URL
4. The image preview will appear as you type
5. Click **"Add Service"** to save

### Recommended Free Image Sources

**Pexels** (https://www.pexels.com)
- Search for "nail art", "manicure", "nail salon", etc.
- Right-click any image → Copy image link
- Paste the link in the Image URL field

### Editing Service Images

1. Click the **"Edit"** button on any service
2. The current image will appear in the preview
3. Change the URL in the **Image URL** field to a new one
4. Click **"Update Service"** to save

### Where Images Appear

- **Homepage** - Services display with their images in a professional grid
- **Service Cards** - Each service shows an image that zooms on hover
- **Admin Dashboard** - Preview your images before saving

### Tips for Best Results

✅ Use **square or landscape images** for best appearance
✅ **Free sources** like Pexels have no licensing restrictions
✅ Images are **cached by browsers** for fast loading
✅ Broken image links will automatically hide the image
✅ If an image doesn't load, try copying the link again and refreshing

### Mobile Responsiveness

All images are automatically optimized for mobile devices, so they'll look great on phones and tablets!
